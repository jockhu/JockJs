JOCKJS - An Smart JavaScript Library
=======================================


 简介
--------------------------------------

此框架吸取了多款主流框架的思想，使用灵活，迷你，核心代码只有2.9K，服务端采用Node架构，实现了对DOM常用的操作方法以及事件、Ajax等封装，拥有模块依赖管理，动态按需载入，在线压缩，模块请求自动合并等。因模块的打包压缩都是在请求的过程中动态完成，所以发布前不需要打包。
配合[JockCss](https://github.com/jockhu/jockcss)（css框架）、[JockJsComponents](https://github.com/jockhu/jockjs-components)（组件库）、[JockJsBusiness](https://github.com/jockhu/jockjs-business-mobile)（业务组件）等组成前端解决方案，拥有丰富的组件库，极大的提升了前端开发效率，完善的性能监控、错误日志监控、事件追踪等支持，为性能、系统稳定性提供了很多保障。目前各个事业部都在使用这套解决方案
